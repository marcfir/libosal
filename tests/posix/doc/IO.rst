Functional Tests
================
  
IOFunction, PrintF
------------------

Tests `osal_printf()` by printing a message
to the terminal (no automated test of output).

IOFunction, PutS
----------------

Tests `osal_puts()` by printing a string to the terminal.

IOFunction, vfprintf
--------------------

Tests `osal_vprintf()` by printing something to the terminal.
